# DETT
Digital Esir Time Table

##Créer un nouveau dépôt
  - git init
  
##Cloner un dépôt existant
  - git init %URL%
  
##Récupérer le dépôt
  - git clone %URL%

##Modifier le code et effectuer des commits
  1. Indiquer les fichiers que vous avez modifié
    - git status
  2. Voir ce que vous avez changé
    - git diff
